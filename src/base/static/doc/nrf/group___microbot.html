Detailed Description

Microbot Service module. This module implements the Microbot Service with the Microbot characteristic. During initialization it adds the Microbot Service and Microbot characteristic to the BLE stack database. Optionally it can also add a Report Reference descriptor to the Microbot characteristic (used when including the Microbot Service in the HID service).

Author
Naran.inc
Note
Attention! To maintain compliance with Nordic Semiconductor ASA Bluetooth profile qualification listings, this section of source code must not be modified.

◆ gadget_return_type_t

enum gadget_return_type_t

Microbot NRF APIs Return Codes.

Author
Naran.Inc
See also
you can receive MIB_SUCCESS when any action succeeded, otherwise refer to the below code
Enumerator
MIB_SUCCESS 

Success

MIB_OTP_FAILED 

OTP code is not correct

MIB_WRONG_INIT_VALUES 

This is the result of checking the validity of the mib_init_t parameter of mib_inti ().

MIB_PSTORAGE_INVALID_BLOCK_NUMBER 

This means that when attempting to access the wrong block number when using Pstorage. Only 1 to 11 are allowed.

MIB_PSTORAGE_INVALID_OFFSET_OR_SIZE 

When using Pstorage, one block provides 80 bytes. offset + size cannot exceed 80.

MIB_REQ_ID_MATCH_FAILED 

This happens when an invalid request ID is received at the endpoint call.

MIB_SEQENCE_FAILED 

This happens when multiple packets are not received sequentially.

MIB_DATA_LENGTH_OVERFLOW 

This happens when there are more packets actually received than the data declared in the Header.

MIB_DATA_CRC_CHK_ERROR 

This occurs when the CRC of the received Header + Body does not match the CRC of the Header.

MIB_INCORRECT_EPID 

This occurs when an undefined endpoint ID is requested.

MIB_REQUEST_RETURN_TIMEOUT 

This happens when the timeout defined in MibCloud is exceeded. You must use mib_return, mib_dynamic to avoid timeout.

MIB_REQUEST_BUF_FULL 

This occurs when the endpoint call request queue received by the microbot is full.

MIB_HAS_NOT_REQUEST 

This is returned when there is no endpoint call request stored at the time of calling '8. APIs mib_request_action'.

MIB_NOTIFICATION_BUFFER_FULL 

This happens when mib_return, event_write is called and there is no space in the transmit wait queue.

MIB_ALREADY_PAIRED 

This happens when the microbot has already done a custom pair and then paired again.

MIB_BUSY 

This happens when the Microbot can not perform other tasks.

MIB_NOT_FOUND_REQUST_ID 

This happens when the request is not found at the time of performing mib_return by request.

MIB_UNKNOWN_ERROR 

If an unknown error occurs

MIB_PACKET_TIMEOUT 

If no more data arrives in the middle of packet reception

MIB_UNAUTHORIZED 

This happens when ‘Connect’ api has different secret key

MIB_CONNECT_TIMEOUT 

If you try BLE connect, you must call the Connect API within 10 seconds. This error occurs after 10 seconds have elapsed.

MIB_PAIR_TIMEOUT 

The whole process of the custom pair should be done in 90 seconds. This error occurs when the time exceeds 90 seconds.

MIB_PACKET_RECV_TIMEOUT 

A continuous Ble packet should always arrive within 2 seconds of the next packet. This error occurs when two seconds have elapsed while receiving a packet.

MIB_MEMORY_ALLOC_FAILED 

Microbot-lib handles Malloc-free for all requests. If this error occurs, this is proof that the firmware no longer has the memory to allocate. Please inspect and search for memory leak.

MIB_INIT_INVALID_EVENTHANDELR 

This error occurs when the evt_handler function pointer of the Mib_init structure is not registered.

MIB_INIT_INVALID_LEDPINNUMBER 

This error occurs when the led number of the mib_init structure is not registered.

MIB_INIT_INVALID_BTNPINNUMBER 

This error occurs when the Button number of the Mib_init structure is not registered.

MIB_INIT_INVALID_MINBATT 

This error occurs when the min battery level of the mib_init structure is not registered.

MIB_INIT_INVALID_MAXBATT 

This error occurs when the max battery level of the mib_init structure is not registered.

MIB_INIT_INVALID_BATTPINNUMBER 

This error occurs when the ADC IN PIN NUMBER of the mib_init structure is not registered.

MIB_INIT_UNDEFINE_ENABLE_WATCHDOG 

This error occurs when the whether or not to use watchdoc of the mib_init structure is not registered.

MIB_INIT_INVALID_ENDPOINT_LENGTH 

This error occurs when the endpoint length of the mib_init structure is not registered.
